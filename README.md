# 1st Year Student Guide Website
This project is a simple and interactive website made to help **1st year engineering students** find their **subjects, syllabus, books, and learning resources** in one place.
It is built using only **HTML, CSS, and JavaScript**, making it perfect for students to guide about the topics of first year.

##  Features
-  Semester-wise subject listing (Semester 1 & Semester 2)
-  Search bar to quickly find subjects
-  Clickable subject cards to view syllabus, books, and links
-  Sticky navigation bar
-  Clean dark-theme UI with hover effects
-  Responsive design for mobile and desktop
-  Footer with About, Quick Links, and Developer details

##  Technologies Used
- **HTML5** – Website structure  
- **CSS3** – Styling, layout, and animations  
- **JavaScript** – Interactivity:
  - Semester toggle
  - Expand/collapse subject cards
  - Search functionality
  - Smooth scrolling

No external libraries or frameworks are used.
